Documentación del Programa de Conversión de Archivos
Descripción
El Programa de Conversión de Archivos es una herramienta que permite convertir entre diferentes formatos de archivos, como Word y PDF. Proporciona funciones para convertir de Word a PDF, combinar varios archivos PDF en uno solo, comprimir archivos PDF y convertir de PDF a Word.
Funcionalidades
El programa tiene las siguientes funcionalidades principales:
1. 
Convertir de Word a PDF: Permite convertir archivos de Word (.docx) a formato PDF (.pdf).
2. 
Combinar archivos PDF: Permite combinar varios archivos PDF en uno solo.
3. 
Comprimir archivos PDF: Permite reducir el tamaño de un archivo PDF mediante la compresión de imágenes y optimización del contenido.
4. 
Convertir de PDF a Word: Permite convertir archivos PDF a formato Word (.docx).
Uso
El programa se ejecuta desde la línea de comandos y ofrece las siguientes opciones:
•
`w`: Convertir de Word a PDF. El usuario debe proporcionar la ruta del archivo de Word a convertir.
•
`c`: Combinar archivos PDF. El usuario selecciona los archivos PDF a combinar mediante un cuadro de diálogo.
•
`p`: Comprimir archivo PDF. El usuario selecciona el archivo PDF a comprimir mediante un cuadro de diálogo y define la calidad de compresión.
•
`r`: Convertir de PDF a Word. El usuario selecciona el archivo PDF a convertir mediante un cuadro de diálogo.
Requisitos del Sistema
El programa requiere los siguientes componentes y bibliotecas:
•Python 3.x
•Bibliotecas externas: `PyPDF4`, `docx2pdf`, `pdf2docx`
Instalación
1. 
Asegúrate de tener instalado Python 3.x en tu sistema.
2. 
Instala las bibliotecas externas ejecutando los siguientes comandos en la terminal:
Copy code
pip install PyPDF4
pip install docx2pdf
pip install pdf2docx
3. 
Descarga el código fuente del programa y guárdalo en tu sistema.
Ejecución
Para ejecutar el programa, abre una terminal y navega hasta el directorio donde se encuentra el archivo `main.py`. Luego, ejecuta el siguiente comando:
css
Copy code
python main.py
Sigue las instrucciones en la terminal para realizar las conversiones y operaciones deseadas.
